# Website for Internet Programming Course Assignments

Welcome to the **Basic Portfolio** repository!  
This repository contains the source code of a simple portfolio website created as part of an **Internet Programming course assignment**.

The website is designed to showcase my profile, skills, and projects in a clean and structured way.

---

## About the Project

This portfolio website serves as a medium for personal branding as well as documentation of my learning journey in web development.

The website includes:
- Experience
- Portfolio
- About Me
- Education Record

The project is built using basic web technologies to keep it lightweight, simple, and easy to understand.
